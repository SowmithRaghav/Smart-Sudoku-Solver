# Smart Sudoku Solver: Matrix Optimization Engine

A high-performance 9x9 matrix solver leveraging recursive backtracking algorithms optimized with low-level bit-masking.

## Tech Stack & Tools
- Language: C++17
- Debugging Suite: GDB Toolchain
- Interface: ANSI Escape Terminal Animations
